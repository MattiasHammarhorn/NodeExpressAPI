Generic "User Manager" project with CRUD operations for a DevSecOps course
To run :
>npm install
>node server.js
>
To run tests:
>npm run test:run (unit tests in vitest) 
>npx playwright test (e2e tests in playwright)
>import 'NodeExpressApi.postman_collection.json' into Postman and run while node server is up (api tests in Postman)
